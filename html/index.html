Introduction

Part 1:
A tool for the construction of a parse tree for a propositional logic formula.
Task 1 is a function to convert the infix propositional logic expression into a postfix propositional logic expression.
Task 2 is a function to convert the postfix expression into a rooted binary parse tree.
Task 3 is a function to traverse the parse tree to output the infix expression back by in-order traversal of the parse tree.

Once the parse tree has been constructed, it can be verifed that the given formula is well formed.

Part 2:
A tool that checks whether the given proof for a sequent is valid or not.
The following proof rules can be checked:

  1. Premise
  2. AND Introduction
  3. AND Elimination
  4. OR Introduction
  5. Implies Elimination
  6. Modus Tollens
